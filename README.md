# Security Intelligence

AI-powered cybersecurity management platform - Your Fractional CISO.

## Features

- **Risk Assessment**: Security risk identification, scoring, and prioritization
- **Compliance Management**: SOC 2, HIPAA, PCI-DSS, ISO 27001, NIST CSF
- **Vulnerability Management**: Risk-based prioritization with SLA tracking
- **Incident Response**: Incident tracking, metrics, and response guidance
- **Threat Intelligence**: Threat landscape analysis and detection
- **Security Posture**: Multi-dimensional security scoring
- **AI Consultation**: Claude-powered CISO consultant

## Tech Stack

- **Backend**: Flask, SQLAlchemy, PostgreSQL
- **AI**: Anthropic Claude API
- **Frontend**: Bootstrap 5
- **Deployment**: Render

## Quick Start

```bash
pip install -r requirements.txt
export ANTHROPIC_API_KEY=your-key
python web/app.py
```
